## v0.1.0 — March 2026

### Added
- Initial Rails port with Ruby 3.2, Rails 7.1
- Custom auth system (email/password, OAuth scaffolding, TOTP 2FA)
- SQLite database with ActiveRecord migrations
- Stripe subscription billing (checkout, webhooks, portal)
- Transactional email via Resend + ActionMailer
- Admin panel (dashboard, user management, analytics, audit logs, blog editor, DB browser)
- User settings (account info, data export, account deletion)
- Blog system with Markdown + frontmatter
- RSS feed and sitemap
- Docker deployment configuration
- Tailwind CSS + Hotwire frontend
