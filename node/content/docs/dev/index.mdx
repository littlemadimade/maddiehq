This wiki serves as the primary knowledge base for anyone (human or AI) working on this codebase. It's designed to be read before diving into source code.

## How to use this wiki

- **Agents**: Read relevant pages here before reading source files. The wiki provides structured context that's more efficient than scanning raw code.
- **Developers**: Update these pages when you add or change features. Documentation is a requirement, not an afterthought.

## Pages

- [[architecture]] -- System architecture, tech stack, and how the pieces fit together
- [[conventions]] -- Coding conventions, patterns, and rules to follow

## Wiki features

- **Backlinks**: Pages that reference this one via `[[wiki-links]]` are shown at the bottom
- **Search**: Use the filter box in the sidebar to find pages quickly
- **Table of contents**: Each page shows its headings on the right for quick navigation
