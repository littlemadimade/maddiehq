Welcome to the MaddieHQ documentation. MaddieHQ is a full-stack SaaS starter kit built with Next.js, SQLite, Better Auth, Stripe, and Resend.

## What's included

- **Authentication** -- Email/password, Google & GitHub OAuth, two-factor authentication
- **Payments** -- Stripe subscriptions with customer portal
- **Email** -- Transactional emails via Resend (welcome, verification, password reset)
- **Admin Panel** -- User management, analytics, database browser, audit logs
- **AI Chatbot** -- Claude-powered chat with voice I/O and file attachments
- **Blog** -- MDX-powered blog with RSS feed
- **Dark Mode** -- System-aware theme switching
