# Hello Maddie

Small Next.js landing page with a lime-and-pink 3D greeting scene built with React Three Fiber.

## Run

1. Install Node.js 18.18+ or 20+.
2. Install dependencies with `npm install`.
3. Start the app with `npm run dev`.
4. Open `http://localhost:3000`.

## Notes

- The 3D text rotates slowly on its own.
- Mouse movement adds subtle parallax and tilt.
- The structure is intentionally minimal so it can grow into a fuller OF management app later.
