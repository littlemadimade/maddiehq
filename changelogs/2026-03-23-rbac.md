## Role-based access control (RBAC)

Granular permission system for admin features.

- Permission constants: `items:create/read/update/delete`, `admin:users/roles/settings/waitlist/logs/analytics/crm/database`
- Admin roles page: create, edit, delete roles with permission checkboxes
- Role assignment to users via admin user detail
- `getUserPermissions()`, `hasPermission()`, `requirePermission()` helpers
- Tables added: `roles`, `user_roles`
